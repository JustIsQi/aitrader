# A股规则与常见坑

## 启用方式

当策略需要遵守 A股个股交易约束时，显式设置：

```python
t.ashare_mode = True
t.ashare_commission = "v2"
```

## 关键约束

- T+1：当日买入的股票不能当日卖出
- 涨跌停：接近或触及涨跌停时，买卖会受限
- 整数手：通常按 100 股为一手
- 手续费：默认推荐 `v2`

这些规则更适合 A股个股，不一定适合 ETF 轮动策略。

## 实施建议

- 需要真实 A股回测时，优先开 `ashare_mode`
- 如果策略逻辑依赖当日买入当日卖出，不要误开 A股模式
- 若资金不足以买入 1 手，订单应直接取消，而不是买零碎股
- 手续费、印花税和撮合约束要在同一套 broker 规则里统一处理

## 与复权和数据的关系

- A股策略的价格判断和排序通常对复权方式敏感
- `qfq` 与 `hfq` 混用时，买卖条件和回测结果会失真
- 排查收益率异常时，先检查 `adjust_type` 是否在策略、loader、pipeline 三处一致

## 调试优先级

出现 A股结果异常时，优先排查：

1. `ashare_mode` 是否真的开启
2. `ashare_commission` 是否符合预期
3. 标的是不是股票而不是 ETF
4. 策略是否隐含了 T+0 假设
5. 复权类型是否一致
