# 短线选股量化分析系统需求文档（V1.0）

### 一、文档目的


明确短线选股量化分析系统的核心功能与规则，支撑 “收盘数据量化处理→开盘人工执行” 的交易模式，确保开发聚焦核心、无冗余功能。

### 二、核心目标



1. 每日收盘后 1 小时内，输出「强势板块名单 + 板块内标的优先级列表 + 交易规则参数」

2. 支持开盘前 10 分钟快速核对，仅保留 “量化阈值触发 + 人工简单判断” 的执行环节

3. 量化规则可配置、可回测，适配短线（1-3 日）交易场景

### 四、核心功能需求

#### 1. 板块量化筛选（优先级：P0）



* 规则配置：支持设置以下指标阈值（可后台调整）


  * 资金阈值：近 1 日主力净流入≥3 亿 且 成交额放量率（当日 / 近 5 日均值）≥1.5

  * 技术阈值：板块指数站上 5+10 日线（均线多头） 且 RSI∈\[30,70]

  * 情绪阈值：当日涨停家数≥3 或 近 3 日涨跌幅排名稳居市场前 10

* 输出：每日筛选出前 5 名的强势板块，按 “资金强度 + 技术共振度” 排序

#### 2. 个股量化选股（优先级：P0）



* 筛选范围：仅从 “强势板块名单” 中筛选个股

* 核心规则（二选一，可配置切换）


  * 追涨策略：收盘价创近 5 日新高 + 当日涨停 / 连板（1-2 板） + 主力净流入≥5000 万 + 量比≥1.2

  * 低吸策略：收盘价偏离 5/10 日线≤3% + MACD 金叉 + 成交额≤近 5 日均值 0.8 + 所属板块资金净流入

* 风险过滤：自动排除 “业绩亏损 + 减持公告 + 停牌预警 + 流通盘＜50 亿 /＞500 亿” 的个股

* 输出：每板块前 3-5 名标的，按 “资金净流入占比 + 封单率（连板股）” 排序，生成标的优先级列表

#### 3. 交易规则量化计算（优先级：P0）



* 仓位计算：


  * 单板块仓位≤30%，单标的仓位≤15%，账户总仓位≤70%

  * 板块优先级 1-2 名：单标的配 8%-10% 仓位；3-5 名：配 5%-8% 仓位

* 止损 / 止盈计算：


  * 止损位：min (前一日收盘价 - 3%，5 日线价)

  * 止盈位：max (近 10 日高点，当日收盘价 + 10%)（可配置梯度止盈：10% 止盈 50%，20% 清仓）

* 开盘触发阈值：


  * 高开幅度 3%-5%，封单量≥流通盘 1%，竞价成交额≥500 万

#### 4. 结果输出（优先级：P0）



* 输出形式： “每日操作清单” 写入postgres，包含以下字段：

| ---- | ---- | ---- | ------------- | ---- | --- | --- | --------------------- |
|      |      |      |               |      |     |     |                       |

- 给出一个每日信号生成的脚本(即“每日操作清单”)


#### 5. 回测功能（优先级：P1）



* 支持导入近 1-3 年历史收盘数据，按配置的规则回测

* 输出回测报告：胜率、盈亏比、最大回撤、单笔平均收益、无效场景统计

### 六、开发边界



1. 仅负责收盘后量化分析与结果输出，不涉及实时交易接口对接

2. 开盘人工执行环节，仅提供量化阈值参考，不包含自动下单功能

3. 风险数据仅做基础过滤，不包含深度基本面分析（如财务报表拆解）
