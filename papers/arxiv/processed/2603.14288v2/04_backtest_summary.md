## 策略参数

### 周频版本
- **回测期间**: 2019-01-01 至 2024-12-31
- **股票池**: 全市场（剔除ST、上市不足7年）
- **调仓频率**: 每周
- **持仓数量**: 20 只
- **权重方式**: 等权重

### 买入条件（至少满足2个）
1. 换手率 > 0.3%
2. PE > 0 且 PE < 100
3. 收盘价 > 60日均线
4. 成交量 > 20日均量的30%

### 卖出条件（满足任一即卖出）
1. 换手率 < 0.3%（流动性枯竭）
2. 跌破20日均线8%
3. 20日跌幅超15%

### 复合因子（5个子因子）

```python
# F1: 换手率冲击 (25%)
(turnover_rate - ma(turnover_rate,20)) / ma(turnover_rate,20)

# F2: 换手率动量 (20%)
ma(turnover_rate,5) / ma(turnover_rate,20)

# F3: 平滑换手率冲击 (20%)
(ma(turnover_rate,5) - ma(turnover_rate,20)) / ma(turnover_rate,20)

# F4: 防御性均值回归 (15%)
(ma(close,20) - close) / close

# F5: 持续关注度 (20%)
LOG(ma(turnover_rate,20) + 1)
```

## 策略优势

### 1. 数据可得性强
- 仅需日频 OHLCV + 换手率
- 无需高频数据、新闻数据、NLP基础设施
- Wind/iFinD 等数据源均可获取

### 2. 市场适配性好
- A股散户主导（80%+），换手率信号更强
- 换手率水平高（2-10%），信息更丰富
- 投资者注意力效应显著

### 3. 因子逻辑清晰
- 换手率冲击：捕捉异常关注度
- 换手率动量：捕捉关注度加速
- 均值回归：捕捉错误定价修正
- 持续关注度：捕捉长期注意力

### 4. 实施简单
- 因子计算简单，无需复杂模型
- 调仓频率适中（周频/月频）
- 持仓数量合理（15-30只）

## 风险提示

### 1. 市场风险
- **熊市表现**: 换手率因子在熊市可能失效
- **极端行情**: 涨跌停板可能导致无法成交
- **流动性风险**: 小市值股票可能面临流动性不足

### 2. 策略风险
- **拥挤交易**: 如果策略被广泛使用，超额收益会衰减
- **因子衰减**: 换手率因子可能随时间衰减
- **参数敏感**: 策略对参数选择较敏感

### 3. 实施风险
- **交易成本**: A股交易成本较高（佣金+印花税 ~0.13%）
- **冲击成本**: 大资金可能面临冲击成本
- **T+1限制**: 当日买入无法卖出

## 优化建议

### 1. 降低换手率
- 增加持仓数量（20 → 30）
- 降低调仓频率（周频 → 月频）
- 增加买入条件数量（2 → 3）

### 2. 增强风控
- 加入行业中性约束
- 加入市值中性约束
- 设置单只股票最大权重上限

### 3. 因子优化
- 加入换手率波动率因子
- 加入换手率偏度因子
- 加入换手率与收益率的交叉项

### 4. 动态调整
- 根据市场环境动态调整因子权重
- 根据波动率动态调整持仓数量
- 根据流动性动态调整换手率阈值
