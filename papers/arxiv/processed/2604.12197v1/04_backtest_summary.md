1. 换手率 > 0.5%
2. PE > 0 且 PE < 100
3. 收盘价 > 60日均线
4. 成交量 > 20日均量的50%

### 卖出条件（满足任一即卖出）

2. 跌破20日均线10%
3. 20日跌幅超20%

### 因子构造（简化版）

```python
# 网络中心性近似因子
# 动量因子（60%）+ 稳定性因子（40%）

signal = (
    normalize_score(roc(close,20)) * 0.60 +           # 20日收益率
    normalize_score(-std(close,20)/ma(close,20)) * 0.40  # 波动率倒数
)
```

**简化理由**:
- 真正的网络中心性需要计算相关性矩阵和图论算法
- 当前回测框架不支持跨股票的相关性计算
- 使用动量+稳定性组合作为近似（高动量+低波动 ≈ 市场领导者）

## 论文原模型 vs 简化实现

### 论文原模型（未实施）

**数学框架**:
```
r_{t+1} = (1-ε)g(r_t) + (ε/N)Cg(r_t)
```

其中:
- `g()`: 非线性迭代函数（如logistic map）
- `C`: 拉普拉斯矩阵的正交变换（定义网络连接）
- `ε`: 耦合参数（控制独立性/依赖性）
- `N`: 集群内元素数量

**实施难点**:
1. 需要求解K×K耦合非线性动力系统
2. 需要定义资产间的连接矩阵（如何定义？）
3. 需要处理混沌系统的数值稳定性
4. 参数选择缺乏指导（ε、N、C如何确定？）
5. 因子缺乏经济学解释

### 简化实现（当前版本）

**核心思想**: 保留网络视角，但用简单指标替代复杂动力系统

**因子逻辑**:
- **动量**: 捕捉市场趋势（类似论文的"共同运动"）
- **稳定性**: 低波动率（类似论文的"中心节点"）
- **组合**: 高动量+低波动 = 稳定的市场领导者 ≈ 高网络中心性

**优势**:
- 实施简单，易于理解
- 有经济学直觉（领导者效应）
- 可快速回测验证

## 附录：完整网络中心性实现示例

如果要实现真正的网络中心性，可以参考以下代码框架：

```python
import networkx as nx
import pandas as pd
import numpy as np

def calculate_network_centrality(returns_df, window=60, threshold=0.3):
    """
    计算股票网络中心性
    
    Args:
        returns_df: DataFrame, 收益率数据 (date × symbols)
        window: int, 滚动窗口
        threshold: float, 相关性阈值
    
    Returns:
        DataFrame, 中心性得分 (date × symbols)
    """
    centrality_scores = []
    
    for i in range(window, len(returns_df)):
        # 1. 计算相关性矩阵
        window_returns = returns_df.iloc[i-window:i]
        corr_matrix = window_returns.corr()
        
        # 2. 构建网络
        G = nx.Graph()
        for sym1 in corr_matrix.index:
            for sym2 in corr_matrix.columns:
                if sym1 != sym2 and corr_matrix.loc[sym1, sym2] > threshold:
                    G.add_edge(sym1, sym2, weight=corr_matrix.loc[sym1, sym2])
        
        # 3. 计算中心性
        try:
            centrality = nx.eigenvector_centrality(G, weight='weight', max_iter=1000)
        except:
            centrality = {sym: 0 for sym in corr_matrix.index}
        
        centrality_scores.append(centrality)
    
    return pd.DataFrame(centrality_scores, index=returns_df.index[window:])
```

这个实现需要在回测框架外部完成，然后将中心性得分作为因子输入。
